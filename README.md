# qb-clean-hud
A hud with a clean interface for QBCore. Inspired by InfamesRP HUD

This script includes:

* Kinematic bars.
* Hide HUD command.
* Custom logo on the top right.
* Compatible with the latest version of QBCore.
* Vehicle mode
* [Coming Soon] Stress System
